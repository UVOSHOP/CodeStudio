Pengertian, Penjelasan, dan Contoh HTML, CSS, dan JavaScript

Pengertian

  • HTML: Bahasa markup untuk membuat struktur halaman web.
  • CSS: Bahasa stylesheet untuk mempercantik halaman web.
  • JavaScript: Bahasa pemrograman agar web jadi interaktif.

Penjelasan dan Contoh

HTML

Membuat struktur web seperti paragraf, tombol, gambar, dsb.

<p>Hello World!</p>

CSS

Mengatur tampilan seperti warna, font, ukuran, dll.

body {
  background-color: #f2f2f2;
  font-family: Arial;
}

JavaScript

Membuat website bisa berinteraksi dengan pengguna.

document.getElementById("btn").addEventListener("click", function() {
  alert("Hello!");
});

Simbol-Simbol Dasar Coding

  • < : Kurang dari, pembuka tag HTML
  • > : Lebih dari, penutup tag HTML
  • / : Garis miring (penutup tag atau direktori)
  • ' / " : Tanda kutip string
  • \\ : Escape character
  • | : Logika OR
  • : : Pisah properti dan nilai di CSS
  • ( ) : Untuk fungsi
  • _ : Pengganti spasi
  • +, -, = : Operasi

Simbol Lainnya

  • # : ID selector CSS
  • . : Class selector CSS
  • [ ] : Atribut selector CSS
  • { } : Blok kode
  • ; : Mengakhiri perintah JavaScript

Simbol dalam HTML & CSS

  • & : Karakter khusus HTML
  •   : Spasi tetap HTML
  • * : Universal selector CSS
  • ~ : Sibling selector CSS
  • > : Child selector CSS
