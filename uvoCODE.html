Pengenalan Komprehensif HTML

Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. Ini bukanlah bahasa pemrograman dalam artian memiliki logika komputasi, melainkan sebuah bahasa markup. Tugas utama HTML adalah memberikan struktur dan makna pada konten di halaman web. Bayangkan HTML sebagai kerangka atau fondasi sebuah rumah; ia menentukan di mana letak pintu, jendela, ruangan, dan elemen struktural lainnya.

Dengan HTML, Anda mendefinisikan berbagai elemen seperti judul, paragraf, gambar, tautan (link), tabel, formulir, dan banyak lagi. Browser web kemudian membaca file HTML ini dan merender (menampilkan) kontennya kepada pengguna sesuai dengan markup yang telah didefinisikan.

Fitur Utama HTML:

  • Struktur Dokumen: Mendefinisikan hierarki dan tata letak konten.
  • Hyperlink: Memungkinkan navigasi antar halaman web atau bagian dalam halaman yang sama.
  • Multimedia: Mendukung penyematan gambar, audio, dan video.
  • Formulir: Memfasilitasi interaksi pengguna melalui input data.
  • Semantik: Memberikan makna pada konten (misalnya, <article>, <nav>, <footer>), yang penting untuk SEO dan aksesibilitas.

Struktur Dasar Dokumen HTML

Setiap dokumen HTML mengikuti struktur dasar tertentu yang harus dipatuhi agar valid dan dapat diinterpretasikan dengan benar oleh browser.

Contoh: Struktur Dasar HTML5
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Judul Halaman Saya</title>
    <!-- CSS, Font, atau script lain bisa ditambahkan di sini -->
  </head>
  <body>
    <!-- Konten yang terlihat oleh pengguna ada di sini -->
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>
Coba Sendiri

Penjelasan Elemen Struktur Dasar:

  • <!DOCTYPE html>: Deklarasi ini mendefinisikan bahwa dokumen ini adalah HTML5. Ini adalah informasi penting untuk browser agar merender halaman dalam mode standar.
  • <html lang="id">: Elemen root (akar) dari setiap halaman HTML. Atribut lang="id" menandakan bahwa bahasa utama konten adalah Bahasa Indonesia.
  • <head>: Berisi meta-informasi tentang dokumen HTML yang tidak ditampilkan secara langsung di halaman. Ini termasuk judul halaman (title), set karakter, link ke file CSS, script, dan metadata lainnya.
    • <meta charset="UTF-8">: Menentukan set karakter yang digunakan, UTF-8 adalah standar yang paling umum dan mendukung berbagai macam karakter.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Mengkonfigurasi viewport agar halaman web responsif terhadap berbagai ukuran layar perangkat.
    • <title>Judul Halaman Saya</title>: Menentukan judul halaman yang akan muncul di tab browser atau judul jendela.
  • <body>: Berisi semua konten yang akan ditampilkan kepada pengguna, seperti teks, gambar, tautan, tabel, dll.

Elemen HTML Umum

Elemen HTML adalah blok bangunan halaman web. Sebuah elemen biasanya terdiri dari tag pembuka, konten, dan tag penutup. Contoh: <p>Ini konten paragraf.</p>. Ada juga elemen kosong (empty elements) yang tidak memiliki tag penutup, seperti <img src="gambar.jpg" alt="deskripsi"> atau <br>.

Heading dan Paragraf

Heading digunakan untuk menandai judul dan subjudul dalam konten. HTML menyediakan enam level heading, dari <h1> (paling penting) hingga <h6> (kurang penting). Paragraf (<p>) digunakan untuk teks biasa.

Contoh: Heading dan Paragraf
<h1>Judul Utama Halaman</h1>
<p>Ini adalah pengenalan singkat tentang topik utama.</p>

<h2>Sub Judul Pertama</h2>
<p>Paragraf ini menjelaskan lebih detail tentang sub judul pertama.</p>

<h3>Detail Lebih Lanjut</h3>
<p>Informasi spesifik di bawah sub judul.</p>

Tautan (Links) dan Gambar (Images)

Tautan (<a>) digunakan untuk menghubungkan ke halaman lain atau sumber daya. Atribut href menentukan URL tujuan. Gambar (<img>) digunakan untuk menyematkan gambar. Atribut src menentukan sumber gambar dan alt memberikan teks alternatif.

Contoh: Tautan dan Gambar
<!-- Tautan ke situs eksternal -->
<a href="https://www.google.com">Kunjungi Google</a>

<!-- Tautan ke bagian lain di halaman ini (membutuhkan ID target) -->
<a href="#css-pengenalan">Pelajari CSS</a>

<!-- Menyematkan gambar -->
<img src="logo.png" alt="Logo Perusahaan" width="100" height="50" />

List (Daftar)

HTML menyediakan beberapa jenis list untuk menampilkan daftar item:

  • Unordered List (<ul>): Daftar item tanpa urutan tertentu, biasanya ditampilkan dengan bullet point. Setiap item didefinisikan dengan tag <li>.
  • Ordered List (<ol>): Daftar item dengan urutan tertentu, biasanya ditampilkan dengan angka atau huruf. Setiap item juga didefinisikan dengan tag <li>.
  • Description List (<dl>): Daftar istilah dan deskripsinya. Menggunakan tag <dt> untuk istilah dan <dd> untuk deskripsi.
Contoh: Unordered dan Ordered List
<h4>Daftar Buah Favorit (Unordered):</h4>
<ul>
  <li>Mangga</li>
  <li>Apel</li>
  <li>Jeruk</li>
</ul>

<h4>Langkah-langkah Membuat Kopi (Ordered):</h4>
<ol>
  <li>Siapkan cangkir dan kopi bubuk.</li>
  <li>Masukkan kopi ke cangkir.</li>
  <li>Tuang air panas.</li>
  <li>Aduk rata dan sajikan.</li>
</ol>

Atribut HTML

Atribut memberikan informasi tambahan tentang elemen HTML. Atribut selalu ditempatkan di dalam tag pembuka dan biasanya datang dalam pasangan nama/nilai seperti: nama="nilai".

Contoh Atribut Umum:

  • id: Menentukan ID unik untuk sebuah elemen (digunakan oleh CSS dan JavaScript).
  • class: Menentukan satu atau lebih nama kelas untuk sebuah elemen (digunakan oleh CSS dan JavaScript).
  • style: Digunakan untuk menerapkan styling CSS inline pada elemen.
  • href: (pada <a>) Menentukan URL tujuan tautan.
  • src: (pada <img>, <script>) Menentukan sumber file.
  • alt: (pada <img>) Menyediakan teks alternatif untuk gambar jika tidak dapat ditampilkan.
  • disabled: (pada elemen form) Menonaktifkan elemen input.
  • placeholder: (pada elemen input) Memberikan petunjuk singkat di dalam field input.
Contoh: Penggunaan Atribut
<p id="intro" class="highlight-text main-paragraph">Paragraf dengan ID dan Class.</p>
<img src="logo.png" alt="Logo UVOcode" style="width:150px; border:1px solid black;" />
<input type="text" placeholder="Masukkan nama Anda" disabled />

Pengenalan Komprehensif CSS

Apa itu CSS?

CSS adalah singkatan dari Cascading Style Sheets. CSS adalah bahasa stylesheet yang digunakan untuk mendeskripsikan presentasi (tampilan dan format) dari dokumen yang ditulis dalam bahasa markup seperti HTML. CSS memisahkan konten (HTML) dari presentasi, yang membawa banyak keuntungan:

  • Pemeliharaan Lebih Mudah: Perubahan desain dapat dilakukan di satu file CSS dan akan berlaku untuk banyak halaman HTML.
  • Konsistensi: Tampilan yang konsisten di seluruh situs web.
  • Aksesibilitas: Memungkinkan pengguna untuk menyesuaikan tampilan sesuai kebutuhan mereka.
  • Performa: Mengurangi duplikasi kode styling di setiap halaman HTML, sehingga ukuran file lebih kecil dan waktu muat lebih cepat.

Dengan CSS, Anda dapat mengontrol warna, font, ukuran teks, spasi, layout (tata letak), background, border, dan banyak aspek visual lainnya dari elemen HTML.

Sintaks Dasar CSS dan Selector

Aturan CSS terdiri dari dua bagian utama: selector dan blok deklarasi.

Selector menunjuk ke elemen HTML yang ingin Anda gayai. Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma (;). Setiap deklarasi terdiri dari properti CSS dan nilainya, dipisahkan oleh titik dua (:).

Contoh: Sintaks CSS
/* Ini adalah komentar CSS */
p { /* 'p' adalah selector (menargetkan semua elemen <p>) */
  color: blue;         /* Properti: color, Nilai: blue */
  font-size: 16px;       /* Properti: font-size, Nilai: 16px */
  line-height: 1.5;
}

.highlight { /* '.highlight' adalah class selector */
  background-color: yellow;
  font-weight: bold;
}

#header-utama { /* '#header-utama' adalah ID selector */
  padding: 20px;
  border-bottom: 1px solid #ccc;
}

Jenis-jenis Selector Umum:

  • Selector Elemen/Tipe: Memilih elemen berdasarkan nama tagnya (misalnya, h1, p, div).
  • Selector Class: Memilih elemen berdasarkan atribut class-nya (diawali dengan titik, misalnya, .nama-kelas). Satu elemen bisa memiliki banyak kelas, dan satu kelas bisa digunakan oleh banyak elemen.
  • Selector ID: Memilih satu elemen unik berdasarkan atribut id-nya (diawali dengan tanda pagar, misalnya, #id-unik). ID harus unik dalam satu halaman.
  • Selector Atribut: Memilih elemen berdasarkan atribut atau nilai atributnya (misalnya, [type="text"]).
  • Selector Universal (*): Memilih semua elemen.
  • Selector Kombinasi (Descendant, Child, Adjacent Sibling, General Sibling): Menggabungkan selector untuk menargetkan elemen berdasarkan hubungannya dengan elemen lain.
  • Pseudo-class: Memilih elemen berdasarkan state tertentu (misalnya, :hover, :first-child).
  • Pseudo-element: Memilih dan menggayai bagian tertentu dari elemen (misalnya, ::before, ::first-line).

Pengenalan Komprehensif JavaScript

Apa itu JavaScript?

JavaScript (sering disingkat JS) adalah bahasa pemrograman tingkat tinggi, dinamis, dan seringkali diinterpretasikan yang paling dikenal sebagai bahasa skrip untuk halaman web. Namun, penggunaannya telah berkembang pesat dan kini juga digunakan di banyak lingkungan non-browser berkat Node.js.

Bersama HTML dan CSS, JavaScript adalah salah satu dari tiga teknologi inti World Wide Web:

  • HTML menyediakan struktur konten.
  • CSS mengatur presentasi (tampilan) konten.
  • JavaScript menambahkan interaktivitas dan perilaku dinamis pada halaman web.

Kemampuan Utama JavaScript di Browser:

  • Manipulasi DOM (Document Object Model): Mengubah konten, struktur, dan gaya halaman HTML secara dinamis.
  • Event Handling: Merespons aksi pengguna seperti klik mouse, input keyboard, atau perubahan halaman.
  • Validasi Formulir: Memeriksa input pengguna sebelum dikirim ke server.
  • Animasi: Membuat efek visual dan animasi.
  • AJAX (Asynchronous JavaScript and XML): Memuat data dari server secara asynchronous tanpa perlu me-refresh seluruh halaman.
  • Penyimpanan Sisi Klien: Menggunakan Web Storage (localStorage, sessionStorage) atau IndexedDB untuk menyimpan data di browser pengguna.
  • Dan masih banyak lagi, seperti bekerja dengan API browser, grafis (Canvas, WebGL), audio/video, dll.

Sintaks Dasar JavaScript

JavaScript memiliki sintaks yang mirip dengan bahasa pemrograman lain seperti Java atau C++. Berikut beberapa aspek dasar:

  • Case-Sensitive: JavaScript membedakan huruf besar dan kecil (variabel dan Variabel dianggap berbeda).
  • Statement (Pernyataan): Instruksi JavaScript biasanya diakhiri dengan titik koma (;), meskipun dalam banyak kasus bersifat opsional karena adanya Automatic Semicolon Insertion (ASI). Namun, praktik terbaik adalah selalu menggunakannya.
  • Komentar:
    • Komentar satu baris diawali dengan //.
    • Komentar beberapa baris diawali dengan /* dan diakhiri dengan */.
  • Blok Kode: Dikelompokkan dalam kurung kurawal { }, misalnya dalam fungsi atau pernyataan kondisional.
Contoh: Sintaks Dasar JavaScript
// Ini adalah komentar satu baris
let message = "Halo UVOcode!"; // Deklarasi variabel dengan let
const PI = 3.14;         // Deklarasi konstanta

/*
  Ini adalah
  komentar
  multi-baris.
*/

function sapa(nama) {
  // Ini adalah blok kode di dalam fungsi
  console.log("Halo, " + nama + "!");
  return "Pesan terkirim";
}

sapa("Pengguna"); // Memanggil fungsi

if (PI > 3) {
  console.log("PI lebih besar dari 3");
}

Daftar Simbol Umum dalam Coding

Memahami simbol-simbol dasar adalah kunci untuk membaca dan menulis kode. Berikut adalah beberapa simbol yang sering ditemui dalam HTML, CSS, dan JavaScript, beserta penjelasannya.

Simbol Dasar dan Umum

< >
Kurung Siku (Angle Brackets): Digunakan dalam HTML untuk mendefinisikan tag. Contoh: <p> (tag pembuka paragraf), </p> (tag penutup paragraf).
/
Garis Miring (Slash):
  • Dalam HTML, digunakan pada tag penutup (misalnya, </body>) atau pada elemen kosong (self-closing tag) seperti <img src="..." /> (opsional di HTML5 untuk beberapa elemen).
  • Dalam path URL atau direktori file (misalnya, images/logo.png).
  • Dalam JavaScript, digunakan untuk operator pembagian (misalnya, 10 / 2) dan untuk mengawali dan mengakhiri Regular Expressions (misalnya, /pattern/g).
  • Dalam CSS dan JavaScript, digunakan dalam komentar multi-baris (/* komentar */) dan komentar satu baris (// komentar di JS, atau bagian dari /* */ di CSS).
' "
Tanda Kutip Tunggal dan Ganda (Single and Double Quotes):
  • Dalam HTML, digunakan untuk mengapit nilai atribut (misalnya, <a href="halaman.html">). Konsistensi disarankan.
  • Dalam JavaScript, digunakan untuk mendefinisikan string (misalnya, let nama = "UVOcode"; atau let pesan = 'Halo!';).
  • Dalam CSS, digunakan untuk nilai atribut tertentu (misalnya, font-family: 'Arial', sans-serif;) atau dalam content property.
\\
Backslash (Escape Character):
  • Dalam string JavaScript, digunakan untuk "meloloskan" karakter khusus. Misalnya, 'Dia berkata, \"Halo!\"' untuk menyertakan kutip ganda di dalam string yang diapit kutip tunggal, atau \n untuk baris baru.
  • Dalam Regular Expressions, digunakan untuk meloloskan karakter meta atau menandakan sekuens khusus.
|
Garis Vertikal (Pipe):
  • Dalam JavaScript, digunakan sebagai operator bitwise OR (|) atau logical OR (||).
  • Kadang digunakan dalam konteks template engine atau sebagai pemisah visual dalam teks biasa.
:
Titik Dua (Colon):
  • Dalam CSS, memisahkan properti dari nilainya (misalnya, color: blue;).
  • Dalam CSS, digunakan untuk pseudo-class (misalnya, a:hover) dan pseudo-element (misalnya, p::first-line).
  • Dalam JavaScript, digunakan dalam objek literal untuk memisahkan kunci dari nilai (misalnya, let orang = {nama: "Adi", usia: 30};), dalam label statement, dan dalam operator ternary (kondisi ? nilaiJikaBenar : nilaiJikaSalah).
( )
Kurung Biasa (Parentheses):
  • Dalam JavaScript dan bahasa lain, digunakan untuk mengelompokkan ekspresi guna mengontrol urutan operasi (misalnya, (2 + 3) * 4).
  • Digunakan untuk mendefinisikan dan memanggil fungsi (misalnya, function sapa() {...} dan sapa();). Parameter fungsi juga diletakkan di dalamnya.
  • Digunakan dalam struktur kontrol seperti if (...), for (...), while (...).
  • Dalam CSS, digunakan untuk nilai fungsi (misalnya, rgb(255, 0, 0), url('gambar.png')).
_
Garis Bawah (Underscore): Sering digunakan dalam nama variabel atau fungsi sebagai pengganti spasi untuk meningkatkan keterbacaan (snake_case), misalnya nama_pengguna. Juga umum digunakan untuk menandakan variabel atau properti "privat" secara konvensi dalam JavaScript.
+ - =
Simbol Operasi Aritmatika dan Penugasan:
  • +: Penjumlahan (JS), penggabungan string (JS), adjacent sibling selector (CSS).
  • -: Pengurangan (JS).
  • =: Operator penugasan (JS, misalnya x = 5;). == (loose equality) dan === (strict equality) untuk perbandingan.

Simbol Khusus CSS

#
Tanda Pagar (Hash / Pound Sign):
  • Dalam CSS, digunakan untuk ID selector (misalnya, #header akan menargetkan elemen dengan id="header").
  • Digunakan untuk merepresentasikan warna heksadesimal (misalnya, #FF0000 untuk merah).
  • Dalam URL, menandakan fragment identifier (link ke bagian tertentu halaman, misalnya halaman.html#bagian2).
.
Titik (Period / Dot):
  • Dalam CSS, digunakan untuk class selector (misalnya, .tombol akan menargetkan elemen dengan class="tombol").
  • Dalam JavaScript, digunakan untuk mengakses properti atau metode objek (misalnya, document.getElementById atau mobil.warna).
  • Sebagai bagian dari angka desimal (misalnya, 3.14).
[ ]
Kurung Siku Kotak (Square Brackets):
  • Dalam CSS, digunakan untuk attribute selector (misalnya, input[type="text"] akan menargetkan elemen input dengan atribut type bernilai "text").
  • Dalam JavaScript, digunakan untuk mengakses elemen array berdasarkan indeksnya (misalnya, angka[0]) atau properti objek menggunakan nama properti sebagai string (misalnya, orang["nama"]).
  • Untuk mendefinisikan array literal (misalnya, let warna = ["merah", "hijau", "biru"];).
{ }
Kurung Kurawal (Curly Braces / Brackets):
  • Dalam CSS, digunakan untuk mengapit blok deklarasi yang berisi properti dan nilai (misalnya, p { color: blue; font-size: 16px; }).
  • Dalam JavaScript, digunakan untuk mendefinisikan blok kode (misalnya, dalam fungsi, pernyataan if, loop).
  • Digunakan untuk mendefinisikan objek literal (misalnya, let mobil = {merk: "Toyota", tahun: 2020};).
  • Dalam JavaScript modern (ES6+), digunakan untuk destructuring assignment dan interpolasi string (template literals) `Halo, ${nama}!`.
;
Titik Koma (Semicolon):
  • Dalam CSS, digunakan untuk mengakhiri setiap deklarasi properti-nilai (misalnya, color: red; font-size: 14px;).
  • Dalam JavaScript, digunakan untuk mengakhiri pernyataan (statement). Meskipun opsional dalam banyak kasus karena ASI (Automatic Semicolon Insertion), sangat disarankan untuk selalu menggunakannya demi kejelasan dan menghindari bug potensial.
*
Bintang (Asterisk):
  • Dalam CSS, digunakan sebagai universal selector, yang menargetkan semua elemen (misalnya, * { margin: 0; padding: 0; }).
  • Dalam JavaScript, digunakan sebagai operator perkalian (misalnya, 5 * 3).
  • Dalam JavaScript ES6+, digunakan untuk generator functions (function* namaGenerator() {}) dan sebagai bagian dari sintaks spread/rest.
  • Dalam komentar multi-baris (/* komentar */).
~
Tilde: Dalam CSS, digunakan sebagai general sibling combinator (misalnya, h1 ~ p akan menargetkan semua elemen <p> yang bersaudara (sibling) dan muncul setelah elemen <h1> serta berbagi parent yang sama).
>
Lebih Dari (Greater Than): Selain sebagai operator perbandingan di JavaScript (a > b), dalam CSS digunakan sebagai child combinator (misalnya, ul > li akan menargetkan elemen <li> yang merupakan anak langsung dari elemen <ul>).

Entitas HTML

Entitas HTML digunakan untuk menampilkan karakter yang memiliki makna khusus dalam HTML (seperti < atau >) sebagai teks literal, atau untuk menampilkan karakter yang tidak mudah diketik pada keyboard.

&
Ampersand: Digunakan untuk memulai sebuah entitas HTML. Untuk menampilkan karakter ampersand itu sendiri, gunakan &amp;.
&lt;
Menampilkan karakter < (kurang dari).
&gt;
Menampilkan karakter > (lebih dari).
&nbsp;
Non-Breaking Space: Spasi yang tidak akan pecah ke baris baru oleh word wrap.
&copy;
Menampilkan simbol copyright ©.
&reg;
Menampilkan simbol registered trademark ®.
&euro;
Menampilkan simbol mata uang Euro €.
